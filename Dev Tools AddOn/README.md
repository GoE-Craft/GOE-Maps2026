# GoE Dev Tools

Minecraft Bedrock add-on for developers with essential debugging and testing utilities.

### NOTE: This is still work in progress!

## Features

- **Custom Flight Controller** - Enhanced flight with adjustable speed and direction control
- **Performance Monitor** - TPS counter
- **Debug Commands** - `/biome` and `/structures` for world analysis
- **Diagnostic Wand** - Custom item for entity/block inspection
- **Teleport Wand** - Custom item for fast travel

## Usage

- Enable the add-on in your world's behavior and resource packs
- Enable/disable flight controller for faster fly speed: /flyspeed 0-10
- Enable/disable perf monitor: `/perf`
- Use debug commands: `/biome`, `/structures`
- Diagnostic Wand - right click block/entity to get usefull debug information
- Teleport Wand - right click to teleport to the first block you're looking at or 200 blocks through air
